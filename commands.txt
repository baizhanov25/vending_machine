git config --global user.name >>> для добавления имени пользователя
git config --global user.email >>> для добавления email пользователя
git config --global --list >>> чтобы узнать текущие настройки
git --version >>> для проверки версии гита
git clone <URL> >>> для клонирования репозитория на локальный компьютер
git checkout <ветка> >>> перейти на существующую ветку
git status >>> чтобы узнать статус репозитория
git log >>> для просмотра истории комитов
git log -n 5 >>> для просмотря последних N комитов
nano <файл> >>> для создания или редакторования файла в терминале
Ctrl + O → Enter → Ctrl + X >>> для сохранения и выхода из nano
git add <файл> >>> для добавления изменения
git add . >>> для добавления всех изменений 
git commit -m "Сообщение" >>> создание комита с сообщением
git restore .` / `git checkout -- . >>> отмены изменения в файлах
